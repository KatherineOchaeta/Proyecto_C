# Proyecto C - Conversor de Monedas

## 📖 Descripción
Este proyecto es un **conversor de monedas** desarrollado con lenguaje **C**.  
Permite hacer conversiones entre **Quetzales**, **Dólares estadounidenses** y **Euros**.  
El programa muestra un menú donde el usuario selecciona la conversión deseada e ingresa la cantidad a convertir.

## 🛠️ Prerrequisitos
Antes de ejecutar el proyecto, nos debemos asegurar de tener:
- **Ubuntu** (en máquina virtual)
- **Compilador GCC** instalado
- **Visual Studio Code** conectado a la máquina virtual
